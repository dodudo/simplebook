# simplebook

一个基于SSM搭建类似于简书的网文分享网站

## 主要技术栈

#### 后端

* spring
* spring mvc
* mybatis
* redis，对整个系统列表做了缓存处理，当重复访问数据库后，优先访问缓存

#### 前端

* 前台系统采用layui的一个ui模板搭建
* 后台页面采用x-admin模板编写

#### 数据库采用MySql
